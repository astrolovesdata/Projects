# COVID-19 Global Data Analysis

## Project Overview

This project analyzes global COVID-19 trends using data from **Our World in Data**.  
The goal was to explore infection patterns, mortality impact, and vaccination progress using SQL queries and data aggregation techniques.

The analysis focuses on identifying key global patterns, comparing country-level outcomes, and tracking vaccination rollout over time.

The dataset was queried using **MySQL** and designed to support visualization in **Tableau dashboards**.

## Key Analytical Questions

The analysis was designed to answer several questions about the pandemic:

1. Which countries experienced the highest **infection rates relative to population**?  
2. What was the **case fatality rate** across countries?  
3. How did **global daily cases and deaths evolve over time**?  
4. Which continents experienced the **largest mortality impact**?  
5. How quickly did **vaccination programs roll out worldwide**?

## SQL Analysis Sections

The SQL script performs several analytical steps.

### 1. Case Fatality Analysis

Calculates the percentage of confirmed infections that resulted in death.

```sql
ROUND(
    cd.total_deaths / NULLIF(cd.total_cases,0) * 100,
    2
) AS death_percentage
````

This metric helps compare the **severity of the pandemic across countries**.

---

### 2. Infection Rate Analysis

Identifies countries with the **highest infection levels relative to population size**.

This helps normalize case counts for countries with very different population sizes.

---

### 3. Mortality Impact Analysis

Determines which countries experienced the **largest death counts relative to population**.

This provides insight into **pandemic severity across regions**.

---

### 4. Continent-Level Aggregations

Analyzes death totals at the continent level using continent records available in the dataset.

---

### 5. Global Time Series Trends

Calculates worldwide daily totals for:

* new cases
* new deaths
* daily mortality percentage

This allows the pandemic's **global progression to be analyzed over time**.

---

### 6. Vaccination Progress Tracking

A **window function** is used to calculate the cumulative number of vaccinations administered per country.

```sql
SUM(COALESCE(cv.new_vaccinations,0))
OVER (
    PARTITION BY cd.location
    ORDER BY cd.date
)
```

This creates a **running total of vaccinations**, allowing vaccination rollout to be tracked over time.

---

### 7. Vaccination Progress vs Population

A **Common Table Expression (CTE)** is used to calculate vaccination progress relative to population size.

The analysis computes the percentage of each country's population that has received vaccinations.

## Key Findings

Analysis of the global COVID-19 dataset revealed several notable patterns across countries and continents.

• **Global case fatality rate remained relatively low overall**, with approximately **150 million total cases and 3.18 million deaths**, resulting in an estimated **death rate of about 2.1%** during the period analyzed.

• **Europe and North America recorded the highest total death counts**, indicating a significant early impact of the pandemic in these regions compared with other continents.

• **Infection rates varied significantly across countries when normalized by population size.** The United States and the United Kingdom showed substantially higher infection percentages than countries such as China and India.

• **Infection growth trends accelerated sharply during late 2020 and early 2021**, reflecting the major global waves of COVID-19 transmission observed during that period.

• **Countries experienced very different infection trajectories over time.** The United States and United Kingdom showed rapid increases in infection percentage, while countries like China maintained relatively low infection rates throughout the timeline.
